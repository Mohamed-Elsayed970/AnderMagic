# English Version

**AnderMagic** is a 2D platformer game developed using the **Godot 4.6** engine. The game features engaging gameplay that combines jumping, running, and using magic to defeat enemies and collect coins.

## ğŸ® Game Features

*   **Smooth Movement System:** Includes walking, running, and jumping.
*   **Magical Abilities:** Players can cast magic spells to eliminate enemies.
*   **Multiple Levels:** The game contains several levels (Level 1, 2, 3) with increasing challenges.
*   **Collection System:** Collect coins to increase your score.
*   **User Interface (UI):** Displays the current coin count and score in real-time.
*   **Sound Effects:** Immersive experience with background music and sound effects for movements and magic.

    *   Press **Run** (or `F5`) to start the game.
